Stringed-Instrument Simulation

Simulate vibrating strings using the Karplus-Strong algorithm

Interfaces:
 MusicalString - simulates musical instrument string resonance
 Instrument - methods for playing a musical instrument using notes mapped to char values
 SongPlayer - plays a song whose notes are read from a file

==========

Based on previous works by Kevin Wayne and Stuart Reges

Uses StdAudio and StdDraw from Princeton University
http://introcs.cs.princeton.edu/java/stdlib/
